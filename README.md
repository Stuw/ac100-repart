ac100-repart
============

ac100 repartitioning helper script.
Contains logic to:
- repartition ac100 for u-boot + gpt
- repartition ac100 for fastboot + gpt
- repartition ac100 to use android 4.x + ubuntu

Usage
=====
./main.py

Note: script was checked on python 2.6/2.7
